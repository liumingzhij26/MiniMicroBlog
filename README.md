迷你社交应用
============

这是一个Mini的SNS系统，完全仿造“饭否”来实现，已经初具雏形，已具备的功能

说明
------------

1. 个人信息管理
2. 简单的登陆注册
3. 好友的关注与取消
4. 发布微博信息
5. django自带的后台管理
6. 较为完善的留言管理
7. 加上了发送邮件修改密码
8. 对于随时心情状态的留言
9. 验证码防止垃圾评论
10. 自定义后台管理
